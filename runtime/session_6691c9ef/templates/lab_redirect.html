Module 10: Open Redirect

An open redirect occurs when an application takes a user-provided URL and redirects the user to it without validation. Attackers use this to craft phishing links that appear to come from a trusted domain.

Redirect Test

Click the buttons to see how the redirect behaves. In a real attack, the target URL would be a malicious phishing site.

URL Manipulation

If Vulnerability Mode is ON, any URL in the parameter will be trusted:
